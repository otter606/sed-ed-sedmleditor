This is the README file for the SED-ED SEDML editor.

What's new this version:

New features
1) Can now obtain SBML models from BioModels referenced by a Miriam URN ( needs internet connection ).
2) Elements that have notes have a note icon displayed inside their shape to indicate this.
3) Adding/editing notes is now un-doable.
4) Navigation around the document. 
   For an active editor ,if you click on an element in the project view, it will be highlighted in the graphical editor.

Numerous bug fixes
